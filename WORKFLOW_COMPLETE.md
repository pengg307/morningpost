## 🔑 Token 使用总结

### ✅ 不需要 Token 的步骤（零 Token 模式）

**占比：95% 的步骤**

- 数据获取（API 调用）
- 数据计算（技术指标、统计分析）
- 报告生成（模板填充、字符串拼接）
- 报告审核（完整性检查、格式验证）
- 推送执行（微信发送、日志记录）

**优势：**
- 速度快（总耗时 < 15 秒）
- 成本低（无 API 费用）
- 稳定可靠（不依赖 LLM 响应）

### ⚠️ 需要 Token 的步骤

**占比：5% 的步骤**

- 财经新闻解析（Step 7 晨报）
- 异常数据人工复核（可选）

**优化方案：**
1. **新闻解析**：使用规则引擎替代 LLM（已实现）
2. **异常复核**：仅对置信度低的数据触发 LLM（可选配置）

## 🚨 故障处理

### 数据源故障

1. **检测**：API 返回空或超时
2. **降级**：切换到备用数据源
3. **通知**：发送告警消息
4. **记录**：记录故障时间和持续时间

### 报告生成故障

1. **检测**：模板填充失败
2. **回滚**：使用上一份成功报告
3. **通知**：发送错误详情
4. **修复**：人工介入修复模板

### 推送故障

1. **检测**：微信 API 返回错误
2. **重试**：最多重试 3 次
3. **降级**：保存到本地文件
4. **通知**：发送告警消息
